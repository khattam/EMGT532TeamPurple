## Image Generation Tips:

1. **Generate in batches** - Start with Hero section first (most important)
2. **Iterate** - If colors don't match, ask DALL-E to "make the background darker" or "increase blue accent lighting"
3. **Consistency** - Use similar phrases across prompts for consistent style
4. **Aspect Ratios** - DALL-E 3 supports:
   - Square: 1024x1024
   - Landscape: 1792x1024
   - Portrait: 1024x1792

## After Generation:

1. Save images to: `drowsiness-wearable-landing/public/images/`
2. Name them clearly:
